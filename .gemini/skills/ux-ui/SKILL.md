# UX/UI Design Skill

This skill provides expert guidance for UX/UI design, from research to final implementation.

## Core Competencies

- **User Centered Design (UCD)**: Empathy mapping, persona creation, and user journeys.
- **Visual Design**: Typography, color theory, layout, and grid systems.
- **Design Systems**: Creating and maintaining reusable component libraries.
- **Accessibility (A11y)**: WCAG 2.1 compliance, screen reader support, and keyboard navigation.
- **Interaction Design**: Micro-interactions, animations, and state transitions.
- **Prototyping**: Wireframes, mockups, and interactive prototypes.

## Common Workflows

### Designing a User Flow

1. Define the user's primary goal and entry point.
2. Map the steps (screens/actions) required to reach the goal.
3. Identify edge cases and error states.
4. Refine with low-fidelity wireframes before moving to high-fidelity UI.

### Accessibility Audit

- **Contrast**: Check color contrast ratios for text and UI elements.
- **Navigation**: Ensure the site is fully navigable via keyboard.
- **Semantics**: Use correct HTML tags (`<header>`, `<main>`, `<button>`).
- **Screen Readers**: Add descriptive `aria-label` and `alt` tags.

## Best Practices

- **Consistency**: Stick to established patterns (Material Design, Human Interface Guidelines).
- **Simplicity**: Minimize cognitive load and friction in user tasks.
- **Feedback**: Provide immediate visual/auditory feedback for all user actions.
- **Responsiveness**: Ensure the design works across all screen sizes.
- **Hierarchy**: Use visual hierarchy to guide the user's eye to key actions.
